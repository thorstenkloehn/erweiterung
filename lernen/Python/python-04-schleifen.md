# Lektion 5: Wiederholungen
In Python gibt es `for`- und `while`-Schleifen. Mit `while` wird Code so lange ausgeführt, wie eine Bedingung wahr ist.

Beispiel:
```python
i = 1
while i <= 3:
    print(i)
    i = i + 1
```
