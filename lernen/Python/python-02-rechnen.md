# Lektion 3: Rechnen mit Zahlen
In Python kannst du Zahlen addieren (`+`), subtrahieren (`-`), multiplizieren (`*`) und dividieren (`/`).

Beispiel:
```python
a = 5
b = 3
summe = a + b
print(summe)
```
