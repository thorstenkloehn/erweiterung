# Lektion 6: Arbeiten mit Listen
Listen sind Sammlungen von Elementen. Du kannst sie mit einer `for`-Schleife durchlaufen.

Beispiel:
```python
fruechte = ["Apfel", "Banane", "Birne"]
for frucht in fruechte:
    print(frucht)
```
