# Lektion 4: Falls-Dann-Bedingungen
Mit dem Schlüsselwort `if` kannst du Code nur dann ausführen lassen, wenn eine bestimmte Bedingung erfüllt ist.

Beispiel:
```python
alter = 20
if alter >= 18:
    print("Volljährig")
else:
    print("Minderjährig")
```
