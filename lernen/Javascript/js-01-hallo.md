# Lektion 1: Hallo JavaScript (Node.js)
JavaScript ist die Sprache des Webs, kann aber mit Node.js auch auf dem Computer (Server-seitig) ausgeführt werden. In Node.js nutzen wir `console.log()`, um Text auszugeben.

Beispiel:
```javascript
console.log("Hallo Welt!");
```

## Deine Aufgabe
Schreibe ein Programm, das `"Hallo Node.js"` auf der Konsole ausgibt.

```javascript
// Schreibe hier deinen Code
```
---
solution: "console.log(\"Hallo Node.js\");"
output_expected: "Hallo Node.js"
