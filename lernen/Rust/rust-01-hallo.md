# Lektion 1: Hallo Rust
Rust ist eine moderne Systemprogrammiersprache. Jedes ausführbare Programm beginnt mit einer `main`-Funktion.

Beispiel:
```rust
fn main() {
    println!("Hallo Welt!");
}
```

## Deine Aufgabe
Schreibe ein Programm, das `"Hallo Rust"` auf der Konsole ausgibt.

```rust
// Schreibe hier deinen Code
```
---
solution: "fn main() {\n    println!(\"Hallo Rust\");\n}"
output_expected: "Hallo Rust"
---
